About lightningthieves

lightningthieves is a Litecoin mining group on burnside's pool, operating since mid-April 2013. This site monitors its progress.

This web site is written in node.js, HTML5, CSS, and JavaScript. It utilizes the jQuery, Bootstrap, and socket.io libraries.

I'll eventually put up LTC and BTC donation addresses here, as well as links to this site's source on github.
