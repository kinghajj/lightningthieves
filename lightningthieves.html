About lightningthieves

lightningthieves is a Litecoin mining group on burnside's pool, operating since mid-April 2013. This site monitors its progress.

This web site is written in node.js, HTML5, CSS, and JavaScript. It utilizes the jQuery, Bootstrap, and socket.io libraries.

I'll eventually put up LTC and BTC donation addresses here, as well as links to this site's source on github.

Details

The server periodically fetches from various JSON APIs. When you load this page, a connection between your browser and the server keeps this page up-to-date with the latest data.

You can press the table headings of the exchange rates to change the units displayed in the other tables.

You can press the "Fetch" button on the top to request that the server fetch data from its sources again (though the server prevents you from requesting this too often.) The "Refresh" button updates the page with new data sooner than the usual schedule.
